.grid  // zentriert die Seite und gibt die Breite vor

.col-r-1 // gleichmässige Spalten 1 bis 12 (pro Reihe 12 Spalten)
.col-i-1 // ungleichmässige Spalten 1 bis 12 (pro Reihe 12 Spalten)
Optionen:
-> .center: zentriert die Spalte (bei nur einer Spalte pro Zeile)
-> .right: schiebt die Spalte nach rechts
-> .nested: nach dieser Klasse können weitere Spalten eingesetzt werden (auch mit .center und .right kombinierbar)
